# Umamusume-Grand-Live-Song-Calculator

一个面向《闪耀！优俊少女》（赛马娘 Pretty Derby）  
**「偶像杯」剧本**的网页端辅助计算工具。

本工具用于帮助玩家在养成过程中**合理规划歌曲购买顺序、代币消耗与练习次数**，避免因误判而浪费回合或代币。

---

## ✨ 项目特色

- 📅 **完整时间轴支持**
  - 出道战倒数第八个回合后
  - 第二年一月前半
  - 第二年七月前半
  - 第三年一月前半
  - 第三年七月前半

- 🎵 **歌曲商店模拟**
  - 按时间节点逐步解锁歌曲
  - 支持歌曲购买 / 退款
  - 自动统计「当前时间点已买歌次数」

- 🧮 **代币精确计算**
  - 五种代币实时录入与扣除  
    - 速度（Da）
    - 耐力（Pa）
    - 力量（Vo）
    - 毅力（Vi）
    - 智力（Me）
  - 自动计算：
    - 建议购买歌曲总消耗代币
    - 当前代币 - 建议代币差值（不足标红）

- 📘 **练习表系统**
  - 根据【时间 + 买歌次数】精确计算：
    > 购买下一首歌曲前，还需购买多少次课程
  - Wiki 页面支持按时间筛选查看完整练习表

- 🧠 **卡歌机制支持**
  - 在关键时间节点切换时询问是否「卡歌」
  - 若卡歌：
    - 下一次购买歌曲**不计入当前时间节点买歌次数**
    - 更贴近真实游戏逻辑

- 🏷️ **歌曲优先级与建议标记**
  - 歌曲按优先级排序展示
  - 标注「建议购买」歌曲，便于规划路线

---

## 🖥️ 页面结构说明

### 操作页面
- **左侧：库存区**
  - 当前代币输入
  - 已购买歌曲列表（支持退款）
  - 代币差值提示

- **右侧：歌曲商店**
  - 时间节点按钮
  - 当前时间 / 买歌次数
  - 「购买歌曲前还需购买课程次数」重点提示卡
  - 建议购买歌曲总代币消耗
  - 当前可购买歌曲列表

### Wiki 页面
- **左侧：歌曲库**
  - 展示所有已录入歌曲
  - 完整信息一览（效果 / 代币 / 优先级 / 是否建议）

- **右侧：练习表**
  - 按时间节点筛选
  - 买歌次数 → 所需练习次数对照表

---

## 🚀 使用方式

1. 在浏览器中打开 `index.html`
2. 阅读并确认使用声明
3. 输入你当前拥有的五种代币数量
4. 按实际养成进度点击时间节点
5. 根据提示购买歌曲 / 规划练习
6. 如发生失误，可在库存区点击歌曲进行退款

---

## ⚠️ 注意事项

- 本工具为 **纯前端静态页面**
  - 不读取任何游戏数据
  - 不与服务器交互
  - 不保存用户信息

- 所有数值、规则仅供参考  
  **请以游戏内实际表现为准**

- 「建议购买」标签：
  - 基于 **三速二智一友人** 的常规配卡思路
  - 不同配卡请自行判断取舍
